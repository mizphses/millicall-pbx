SIP電話機登録画面では、電話機が Asterisk に接続するための認証情報（ピア）を管理します。

<Image src={peers} alt="SIP電話機一覧" />

## ピアの追加

1. **新規登録** をクリック
2. 以下を設定:
   - **ユーザー名** — SIP 認証ユーザー名（例: `ext800`）
   - **パスワード** — SIP 認証パスワード
   - **トランスポート** — UDP または TCP
   - **コーデック** — 使用するオーディオコーデック（ulaw, alaw）
3. 保存

保存後、Asterisk の設定が自動的に再生成・リロードされます。

## 電話機の設定

### 自動プロビジョニング（推奨）

[デバイス管理](/millicall-pbx/usage/devices/) で電話機を登録すると、プロビジョニングURLが自動生成されます。電話機にURLを設定するだけで、SIP 認証情報が自動配信されます。

### 手動設定

電話機の SIP 設定画面で以下を入力:

| 項目 | 値 |
|------|-----|
| SIP サーバー | PBX の IP アドレス |
| ポート | 5060 |
| ユーザー名 | ピアのユーザー名 |
| パスワード | ピアのパスワード |
| トランスポート | ピアのトランスポート設定に合わせる |
