ログイン後に表示されるダッシュボードでは、システムの概要を確認できます。

<Image src={dashboard} alt="ダッシュボード画面" />

## 表示される情報

- **内線アカウント数** — 登録されている内線の総数
- **SIP電話機** — 登録されているSIPピアの数
- **外線トランク** — 設定されている外線接続の数
- **接続デバイス** — プロビジョニング済みのデバイス数

## クイック操作

ダッシュボードから直接以下の操作に遷移できます:

- **内線アカウントを追加** → 新しい内線番号の作成
- **SIP電話機を登録** → 新しい SIP ピアの追加
- **デバイス管理** → 電話機デバイスの登録・管理
