外線トランク画面では、SIP プロバイダーとの接続を管理します。トランクを設定すると、外部の電話番号との発着信が可能になります。

<Image src={trunks} alt="外線トランク一覧" />

## トランクの追加

1. **新規登録** をクリック
2. 以下を設定:
   - **名前** — 識別用の内部名（例: `hikari-trunk`）
   - **表示名** — 管理画面上の表示名
   - **ホスト** — SIP プロバイダーのサーバーアドレス
   - **ユーザー名 / パスワード** — プロバイダーから提供された認証情報
   - **DID 番号** — 着信番号（外線番号）
   - **発信者ID** — 発信時に表示される番号
   - **着信先** — 着信時の転送先内線番号
   - **発信プレフィックス** — このトランクを使う発信番号パターン
3. 保存

## 着信ルーティング

トランクの **着信先** に内線番号を設定すると、その番号への着信がルーティングされます。ワークフロー内線を指定すれば、AI が自動応答します。

## 発信ルーティング

**発信プレフィックス** に `0` を設定すると、`0` で始まる番号をダイヤルした際にそのトランクから発信します。複数のプレフィックスはカンマ区切りで指定できます。
