内線アカウント画面では、電話機とワークフローを含むすべての内線を一元管理できます。

<Image src={extensions} alt="内線アカウント一覧" />

## 内線の種類

| 種別 | 説明 |
|------|------|
| **電話** | SIP 電話機に紐づく通常の内線 |
| **ワークフロー** | AI 応対やIVRフローに紐づく内線 |

## 電話内線の追加

1. **内線追加** ボタンをクリック
2. 内線番号（例: `800`）を入力
3. 表示名を設定
4. 接続先のSIPピアを選択
5. 保存

## ワークフロー内線の追加

1. **ワークフロー追加** ボタンをクリック
2. ワークフロー名、内線番号を入力
3. 保存後、ワークフローエディタで応対フローを構築

ワークフローの詳細は[ワークフロー](/millicall-pbx/usage/workflows/)を参照。

## 操作

- **編集** — 内線の設定変更
- **エディタ** — ワークフロー内線の場合、ビジュアルエディタを開く
- **削除** — 内線とワークフローを削除（SIPピアは残ります）
